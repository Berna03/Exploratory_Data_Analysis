## Project Description
Exploratory data analysis of the grocery delivery platform 'Instacart', focused on identifying user shopping patterns.

## Objectives

- Clean and prepare the data for analysis
- Identify shopping patterns by time of day and day of the week
- Analyze the most popular and frequently reordered products
- Study customer repurchase behavior
- Generate insights about consumer habits

## Data Structure
The project work with 5 main datasets.

- 'instacart_orders.csv' - Orders information
- 'products.csv' - Product catalog
- 'order_products.csv' - Products per order
- 'aisles.csv' - Products aisles
- 'departments.csv - Products departments

## Technologies Utilized

- Python
- Pandas - Manipulation and Data analysis
- Numpy - Numeric operations
- Matplotlib - Data Visualization
- Jupyter Notebook - Documentation

## Included Visualizations

- Temporal distribution histograms
- Bar plots of popular products
- Comparative analysis by days of the week
- Purchase behavior distributions
